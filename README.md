# Note-Making Backend
Developed REST API using express.js and used MongodB as database. 

# Live Demo Link : 
Backend : https://notes-making-backend.onrender.com/
<br>
<b> To get all notes </b>
<br>
GET Request : https://notes-making-backend.onrender.com/api/v1/notes/
<br>
<b> To create notes </b>
<br>
POST Request : https://notes-making-backend.onrender.com/api/v1/notes/
<br>
<b>To update notes </b>
<br>
PUT Request : https://notes-making-backend.onrender.com/api/v1/notes/
<br>
<b> To search notes </b>
<br>
GET Request : https://notes-making-backend.onrender.com/api/v1/notes/search
